API Endpoints

1. Analyze File

POST /api/analyze

Upload a file for analysis. Use multipart/form-data with a 'file' field.

curl -X POST -F "file=@/path/to/file.exe" http://localhost:5000/api/analyze
2. Get System Status

GET /api/status

Get status of all available components and real-time monitoring.

curl http://localhost:5000/api/status
3. Start Real-time Monitoring

POST /api/realtime/start

Start real-time monitoring of specified directories.

curl -X POST -H "Content-Type: application/json" -d '{"directories": ["/path/to/watch1", "/path/to/watch2"]}' http://localhost:5000/api/realtime/start
4. Stop Real-time Monitoring

POST /api/realtime/stop

Stop real-time monitoring.

curl -X POST http://localhost:5000/api/realtime/stop
5. Get Real-time Monitoring Status

GET /api/realtime/status

Get status and results of real-time monitoring.

curl http://localhost:5000/api/realtime/status
